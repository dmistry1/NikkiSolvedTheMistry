# Image filenames

Place photos in this folder with these names:

- `img1.JPG` - main couple photo
- `img2.JPG`
- `img3.png`
- `img4.jpeg`
- `img5.jpeg`
- `img6.jpg`
- `img7.jpeg`
- `img8.JPG`
- `img10.png` - men's dress-code example
- `img11.png` - men's dress-code example
- `img12.png` - women's dress-code example
- `img13.png` - women's dress-code example
- `dress-ref.jpg`

You can use `.jpg`, `.JPG`, `.png`, or `.webp`, but update the matching `src="img/..."` path in `index.html` if the filename changes.
